## Overview

This folder contains the code used to compute the analyses and plot the figures presented in the main text and supplementary information. The actual data and large files are **not** included in this code repository, but they can be downloaded from the **Zenodo** (https://doi.org/10.5281/zenodo.17973874) or **DANDI** (https://doi.org/10.48324/dandi.001530/0.260605.0601) repository associated with the publication. The figure folders are also empty and will be filled once the code is run. The Zenodo repository contains reduced versions of the data structures (NWB files with spike times only) while the DANDI repository contains versions with voltage traces included in the files.

## Subfolder contents

### 1. `data_electrophysiology`
NWB files for the **35** high-density extracellular electrophysiology recording sessions.

### 2. `data_optogenetics`
NWB files for the **91** optogenetic inactivation sessions.

### 3. `data_helpers`
Three MATLAB data structures used for the analyses:

- `Area_list.mat` — list of Allen CCF locations for each cortical area of interest
- `Area_Top_Coordinates.mat` — AP/ML coordinates for each cortical area of interest
- `Lick_times.mat` — times (s) of spontaneous licks and continuous jaw position for each high-density extracellular recording session

### 4. `functions`
Seven toolboxes:

- **allenCCF** — registration of probe tracks to the Allen CCF
- **colors** — color maps for figures
- **libsvm-master** — SVM classification and regression
- **NeurodataWithoutBorders-matnwb-5396243** — MATLAB interface for reading and writing Neurodata Without Borders (NWB) files
- **npy-matlab** — read and write NumPy's NPY format (`.npy` files) in MATLAB
- **prettify_matlab-main** — figure design optimization
- **subspaceID-main** — identify null and potent subspaces

Plus custom functions used in the analyses and figure-plotting codes.
